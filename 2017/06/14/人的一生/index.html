人的一生

发表于 2017-06-14 | | 阅读次数
| 字数统计 816 | 阅读时长 3



人生中，我们会经历很多事情。但是有一些事情，是需要我们铭记的。或许，在我看来，人生第一恩便是父母给的，而后，奶奶陪伴着我长大，勤勤恳恳。记忆最清晰的还是，在我上一年级的那段时间，奶奶陪着我，早上天还没亮，就拿着手电筒出发了，到镇上的小卖部吃热包子。或许当时是身在福中不知福。现在回想起来，是多么地美好。不官是百善孝为先，还是作为人的根本。从小到大，父母亲人给的恩德，永远不能忘怀。存在这个世界上，有很多意义，但是最基本，最重要的是，一个好的家庭。一个幸福的家庭。作为儿女，有责任去建设一个和谐的家庭。除了孝顺之外，便是恩，从小到大，由于家庭的贫困，我感受到了许许多多人的恩德，他们对我以及我的家庭的关心和恩德，不离不弃的恩德，这让我难以忘怀，铭记一生，那些可爱的人们，我会永远记住。我慢慢地长大，我也渐渐地感受到人间善良的温暖，这的确让我暖到心窝。不仅仅是亲人关心，在我看来，生活中也存在这那些不是亲人的朋友会贴心地，真心地去关爱你。这让我感受到了亲人般的关心，让我很心暖。
或许人生中的一件很难的事情就是组件属于自己的家庭了吧。或许是我面临的世界太复杂，我们最初感受到那份十分纯正的爱情，似乎总是会受到很多很多外界因素的影响。但是实际上，我们在学会如何生存地更加好，怎么让自己的生活变得更加美好。但是，我们或许会感觉到，当我们渐渐地比原来懂事了，我们也就真正失去那份当初最纯正的爱情，我们有生活地更好的目的，但是，我们没有学会去感受，学会去满足。或许在我看来，最纯正的爱情，只是两个人正真相爱，想在一起，不管什么样的环境，相互鼓励，一起去改变。但是面临现在的社会，以及物竞天择的准则，差的人终究是会被淘汰的。而现实社会中也存在这很多更加好的选择。
再谈谈事业吧，或许是每一个人都会有不同的想法。我对于事业来看，它仅仅是能为家庭带来更好的生活的工具，当然，也是我活的成就感的一个舞台。事业，不可否认是生活的物质基础，有了好的事业才配得上拥有更好的家庭，才配得上拥有更好的事物。但是说到底，或许就是，赚更多的钱，赢更好的生活。
然而，无论，平穷，还是富贵。真情永恒。每次回家，都能感受到许多人的真情，不论是听说，还是亲眼所见。这都让我感动万分。

或许是有了对比，才有了感伤
感恩在于平凡的关心

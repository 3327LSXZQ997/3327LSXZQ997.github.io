感恩在于平凡的关心

发表于 2017-06-15 | | 阅读次数
| 字数统计 922 | 阅读时长 3



善良其实和简单，施恩与人其实也很简单。但是拥有一颗善良的心，有一个善良的信念却是十分困难的。其实吧，施恩者，不会想着有什么回报。而对于报恩者来说，其实也十分简单。因为，施恩者在乎的不是得到多少，而是简简单单地有一颗感恩的心即可。因为，他们懂得，一颗善良的心是最珍贵的，或者说，一颗知恩图报的心是十分珍贵的。
许久没有回家了，也许就没有回家看看那些可爱的乡亲们了。那些在我们家困难的时候，无条件伸手相助的人了。今天回到家，我亲自下厨，喊上那些可爱的人，到我家聚一聚。他们都是淳朴的农村人，他们比任何人都懂得什么是纯真的恩情，什么是人与人之间的帮助。如今，我亲他们来我家吃饭，他们在电话里，直率的地答应，有一股开心的气息，我深深地明白，那不是因为我喊他们吃饭。那是因为，我又一份懂得回报的心情，这让他们感到欣慰，让他们产生共鸣。或许，我这份懂得感恩的心是他们对于我们家的关心所培养的。让我慢慢地懂得，什么是感情，什么是关爱。
或许，有时候在外面呆久了，我却不是太想回家。不是因为，我不想家，是因为，我不知道该什么时候回家，该回家多久。有时候，我却会感觉回家没有什么事情可以做。或许，在很多时候，我错了。什么时候都可以回家，不仅有牵挂我的亲人，还有那一片故土，以及那一片恩情，一份让人心暖的恩情。世界有很多事情我们可以做，而最重要的或许是找到我们生活的寄托，只有我们的生活有了目标，有了寄托，生活才会变得有意义。活着才会是有意义的。而我们活着的目的无非于两点，为了获得更好的物质生活，为了获得更好的精神生活。而精神生活便是感情生活，包括很多种类的感情生活。而真正的感情，确实很难得到的，但是，我们都会感受到。其实最初得到的感情，最初冲我们大脑中发出的感情是最纯真的，没有任何目的性，不会有任何其他的物质性的目的。而当我们生存在这个物质性越来越强的社会，以及我们本身所处的低劣的环境，让我们的心智被蒙蔽了。渐渐地，很多人会开始改变，为了追求更好物质生活而对于感情生活产生错觉。其实，物质生活的追求，并不会影响到感情生活。人的一生本来就可以追求很多东西。但是，往往，我们会面临很多选择性的问题，但是其实，并不需要选择。我们有权利在追求更多的东西。我们不需要战略性的放弃一些东西。因为，当你扔掉了一块石头，再次捡起来了，或许就不是当初那块纯真的石头了。有句话叫做：你永远也不会两次踏入同一条河流。
每个人都有自己的本心，都有自己的生活原则。那么，就让我们学会好好走，我们自己的路。

人的一生

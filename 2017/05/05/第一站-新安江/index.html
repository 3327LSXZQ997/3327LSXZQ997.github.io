第一站----新安江

发表于 2017-05-05 | | 阅读次数
| 字数统计 205 | 阅读时长 1


记得去的前一天晚上，我还准备了一张歌单，准备唱给她听。也是一种小心思吧。之后，我们一起去看了第一场电影。当然，在那段时间，我们一起换了qq头像，一起改了个性签名：缘来如此，事在人为。这个签名一直持续了三年。也见证了我们在一起的三年时光。那天看完电影之后，稍微逛了下超市，我便送她到车站，看着她坐上回家的车。我也独自回家。
后来我们也出到新安江看过机场电影，影响最深的是，新安江的奇雾吧，早晨的时候，我们坐在江边的亭子里，就看着雾气慢慢飘过来，也有一阵阵风，特别有感觉。

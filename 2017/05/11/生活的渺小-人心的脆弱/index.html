生活的渺小----人心的脆弱

发表于 2017-05-11 | | 阅读次数
| 字数统计 758 | 阅读时长 3



或许，忍心重在生活的残酷中不断地成长，而又怎样去执着的心呢？

或许，生活的力量很强大，它能够去改变一切，但是唯一改变不了的确实那一份执着。尽管，心，有时候也是脆弱的，但是那一份执着的信念似乎总能战胜。

又或许，生活本来就没有公平可言，因为那都是相对的，那都是个人的看法罢了。你若是觉得公平，那就公平；觉得不公平，那就，不公平。尽管，我一直相信，生活中会是公平的，只是我没有想到，生活会突然让我失去一些东西，让我措手不及。而，又或许是我做错了什么，得到的惩罚吧。可我却想不到我的所犯之错。

有时候，我也会去思考一些，看似深奥，又或许的确深奥的问题。我为什么而活着。或许，这让我想起了在《成功人士的七个好习惯》里面所得到的：人生在世，有很多种生存的方式，或许说是对于人的一个人类吧：“自我型”、“家庭型”、“事业型”、“爱人型”……而有时候我会思考我自己是哪一种。

或许，我就是所谓的家庭型吧。从小到大，可以说，我是比较幸运的一员。我的家庭也非常和睦，这让我深深受益，我必须要感谢，我家庭里的每一个人对于我的教育。在我的家庭里，几乎没有纷争，这让我深深地感受到，什么是和为贵，什么是谦让。当然，最受影响的是我的父亲，对于我母亲的关怀，那是一种表面上平平淡淡，安安稳稳，而内心充满了包容和爱。这是我能深深体会到的。我父母对于奶奶的细心照顾，让我知道了什么是孝，或许我也懂得了百善孝为先的说法。而我，是从小和奶奶一起生活的，奶奶对于我无微不至的照顾，我今生无法忘怀，同时我也学会什么是感恩，我对于奶奶的关心，不仅仅是孝顺，更是感恩。而我与大我12岁的哥哥，我懂得什么是友谊，年龄相差之大，友谊，情怀却依然深入我心。致此，我真诚地感谢我的家庭带给我的一切，让我学会的一切。

而我此生的梦想就是，建设一个像我成长下的家庭。

人生中，会有很多很美好的东西。遇到了，选择了，就要好好珍惜，好好经营。尽管人生的旅途有很多更好的选择，但是，人生，其实也没有几次选择的机会。选错了，变回失望，失望多了，便被生活吞噬了。往往，自己内心最初的选择，或许是最珍贵的，那是心得感觉。

且行，且珍惜。

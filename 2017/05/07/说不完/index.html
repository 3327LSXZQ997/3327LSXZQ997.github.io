说不完

发表于 2017-05-07 | | 阅读次数
| 字数统计 414 | 阅读时长 1


人生中有许多话是说不完的。特别是身处异地，有些话，想说，说了，却没有用。因为不在身边。尽管我曾经努力去看望她，或许，相见时喜悦，离别时悲伤，经历了一次又一次。尽管我也不是特别坚强，也会落泪，或许只在离别之后远离北京的火车上。来北京时的喜悦，离开时的悲伤，经历了许多。或许15个小时的火车，很难受，我却没有任何感觉，来时因为喜悦，感受不到任何劳累，离别时的难受，劳累又怎能相提并论。或许，在北京，件简简单单的两天，简简单单的陪着她，吃个饭，看个电影，逛个街都会很开心。可是我们一直坚信，能好好地走下去。我们也在qq，微信。时不时秀恩爱，或许是对于千里之外的一个安慰。可是就在这个开学，她身边的人都脱单了，回到寝室和她聊得便是出去玩的事情，她被虐哭了。可我做不了什么，现实好残忍。她想一个人过，没有我，就不会抱着希望。我说过，我去陪她，可她觉得，我去一次也是一次，两次也是两次，又怎么能时时刻刻陪着她呢。或许，我给她的安全，存在感实在太少了。而我只想有人能支持我，支持我找回她。异地恋，也是青春，也是一份感情，我不舍，不想失去。这些文章，我只想在某一天，能带给她感动。

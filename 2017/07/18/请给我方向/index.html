请给我方向

发表于 2017-07-18 | | 阅读次数
| 字数统计 508 | 阅读时长 2



反反复复，悲伤难过了，将近四个月过去了。我想等一个爱我的你回来。可是我却没有什么方向。这四个月，或许有无数次恳求你回到我身边，而你要的是时间，而或许时间是我的伤痛。眼看即将等到你的归来，不知你是累了，还是怎么了，我没有看到你给我的方向，没有看到方向里的你。面对你平淡的语气，难道你还在考虑着什么，我将会不会是你的选择，或许还不确定。我不由得悲伤，难过起来。我只能安慰自己，你累了，在给你点时间，让你回家好好休息几天。面对微信里的你，我有很多话，却不敢说出口。也不敢喊你，或许，也不合适喊你那些，曾经会让我感到幸福的字眼。而现在，我却只能把它放在心里。
这段时间，我一直在想，我拥有什么，以及我将来会拥有什么。我是否有能力创造一个幸福的家庭。或许，面对未来的三年，你在读研，我在上班，又是异地三年，不知你是否会选择我，，还记得刚刚我们相恋时候的你，你说你不在乎异地恋，而现在，或许你无法承受了。
人，总在不断地被比较，我在PK中是否能胜出，或许，我没自信了。一段距离直接把我狠狠滴甩开了，我还有什么优势。我想我有的唯一的优势就是，我在你心里留存的一些回忆。但是，或许，前段时间对你的打扰，磨灭了许多，我不知是否还有一些留存。我或许不该悲观的，可是我又怕我太乐观，从高处摔落，很痛。
我好想好想把你找回，把曾经那个天真的你找回。
而，我，却没有能力去改变。
只求上天，帮我一次吧。

曾经唱过的难听的歌

农历六月初九

发表于 2017-07-01 | | 阅读次数
| 字数统计 366 | 阅读时长 1



再过十几分钟就是一个特殊的日子了。
22年前的今天，你诞生了，那时候我已经有半岁了。
你曾和我说，你小的时候很可爱的。当然，在和我说这句话时候的你，也很可爱。你的生日，希望你能开心。我还依稀记得以前帮你过的生日，虽然，确实有些寒碜了，但是我能感觉到，你也挺开心的。此次的生日，就让她留到以后吧，还是像往常一样，我们一起过。或许，明天，你的生日，我做不了什么，我也只能是给你买了点吃的“北海道白色恋人”，但是，贺卡不是我亲手写的，当然，我答应帮你补上生日，下次，我会好好准备的。虽然不知道是多久之后。
还有就是，希望你能顺利一些吧。这段时间，你一个人，也不知道是不是还是因为你室友天天虐你而不开心，还是，你已经不在去想这些了，还是你已经习惯了，又或者，你忙的没空趋向一切东西。而我呢，我明显地感觉到，生活中冷清了许多，有时候也会找不到东西来埋葬自己。
不管如何把，希望你顺利地走完这一段路。或许，这段路，我不能陪你走吧。
最后，再次把生日祝福给你，从心里面。

梦
